# Store | 線上商店

## 專案說明：

- 網站抓取E-Commerce API來顯示商店看板、分類、產品
- 可透過篩選來呈現符合條件的產品
- 資料皆使用Server action抓取，由伺服器端渲染提高SEO分數
- 使用Zustand管理購物車
- 結帳時使用hook提交訂單資料到E-Commerce

## 開發使用到的技術：

- **語言**:  TypeScript
- **前後端**:  Next.js
- **CSS Style**: Tailwind CSS、shadcn/ui
- **Server Action**
- **Hook**: TanStack Query
- **欄位驗證**: Zod
- **狀態管理**: Zustand
